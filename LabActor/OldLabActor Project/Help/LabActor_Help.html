Spectrum Recorder

Spectrum Recorder module sweeps frequency and records complex response using Red Pitaya. The sweep frequency increment is either uniform and equal to Frequency step, or equal to Frequency step only inside defined focus regions and Frequency step*acceleration factor elsewhere

Line Detector

Line Detector module finds amplitude peaks in spectrum file and fits complex Lorentzian around peaks

Each individual resonance parameters are determined via non-linear curve fit of the signal data in complex representation , where’re  X is “Signal X”, Y is “Signal Y”, M is magnitude,  is phase, and f is frequency. The resonance fit model is , with fit parameters as follows: f0 is center frequency, is resonance half-width, A is amplitude, θ is phase, and  is background signal.

The initial guess for fit parameters can be provided by either a Peak finder or by last fit results obtained in preceding Line Detector processing run shown in Best fit results control. The Best fit results array can be manually edited by overwriting the values or inserting or deleting rows via right-click pop-up menu.  Each separate resonance is fitted within a frequency Fit window centered on its guessed f0.

Line Detector can optionally send the updated the list of focus regions (f0) to Spectrum Recorder, in other words to follow peaks. The update is delayed depending on how full Red Pitaya input buffer is.
